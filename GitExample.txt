- Git Log: 
	git log 
	git log --oneline
	git log -n <limit> : limit 3 lines 
	git log --author="<pattern>" : limit author 
	git log --grep="<pattern>": limit has the string 
	git log <file>: for the file

	Revert the code that changes in local that not commit:	unstaged
		git checkout .
	Revert lại thay đổi (Khi đã thực hiện lệnh git add) staged
		git reset --hard
			
- List which files are staged, unstaged, and untracked
	git status
- Fetch

	git fetch <remote>
		Fetch all of the branches from the repository. This also downloads all of the required commits and files from the other repository.
	git fetch <remote> <branch>
		Same as the above command, but only fetch the specified branch .	
- Add, commit, push
	Add:
		git add [name-file]
		or multi file git add .
		i.e 
			git add README.md
	Commit 
		git commit [name-file] -m [message]
		i.e 
			git commit README.md -m "duoc sua doi"
	Push 
		git push origin [branch-name]
		i.e 
			git push origin master
			
- Pull 
	git pull 
- Branch
	create branch in local: git branch [branch-name] 
	Deleting Branches: git branch -d [branch-name]
